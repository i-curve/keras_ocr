# 简介

用keras实现ocr定位、识别，后端tensorflow.

* 环境 win10 titanx

# 识别
* 数据集链接: https://pan.baidu.com/s/1jJWfDmm 密码: vh8p (中英数300W+,语料不均衡)

* crnn：vgg + blstm + blstm + ctc 

* densenet-ocr ：densent + ctc 

| 网格结构  | GPU | 准确率 | 模型大小 |
| ---------- | -----------| ---------- | -----------|
| crnn | 60ms | 0.972 |  |
| densent+ctc | 8ms | 0.982 | 18.9MB |
